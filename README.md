# movies-explorer-frontend

Дипломная работа по созданию приложения, в которойм пользователи могут искать фильмы и добавлять их в избранное.

### Функционал:

- Верстка на JSX

### Технологии:

- JSX

- HTML5

- CCS3

- Flexbox

- Grid Layout

- Адаптивная верстка (мобильные устройства, планшеты, десктопы)
